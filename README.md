# Projekt Grafiki Komputerowej

## Założenia

Projekt Silnik3D. Wykorzystuje OpenGL, GLFW i GLEW. Projekt ten został stworzony w ramach przedmiotu Grafika Komputerowa.

### Technologie

- OpenGL
- GLFW
- GLEW
- C++
