Konfiguracja Środowiska C++ dla Silnika Gry na Windows

Poniżej znajdziesz kroki do skonfigurowania środowiska do tworzenia silnika gry w języku C++ pod systemem Windows:

  1. Instalacja Visual Studio:

    Pobierz i zainstaluj Visual Studio (zalecana wersja: Visual Studio Community Edition). Podczas instalacji wybierz opcję "Desktop development with C++".

  2. Instalacja Git:

    Pobierz i zainstaluj Git.

  3. Instalacja CMake:

    Pobierz i zainstaluj CMake.

  4. Instalacja GLFW:

    Otwórz terminal Git Bash i wykonaj następujące polecenia:

    git clone https://github.com/glfw/glfw.git cd glfw mkdir build cd build cmake -G "Visual Studio 16 2019" ..

    Upewnij się, że wybrana wersja "Visual Studio" jest zgodna z tą, którą zainstalowałeś. Otwórz projekt w Visual Studio i skompiluj go.

  5. Tworzenie projektu w Visual Studio:

    Otwórz Visual Studio. Wybierz "Create a new project", a następnie "Empty Project". Dodaj plik main.cpp do projektu, w którym będziesz pisał swój kod.

  6. Konfiguracja projektu z GLFW:

    Skopiuj pliki nagłówkowe GLFW (folder include) do folderu C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\VC\Tools\MSVC\14.xx.xx\include. Skopiuj bibliotekę GLFW (plik .lib z folderu lib-vc2019) do folderu C:\Program Files (x86)\Microsoft Visual Studio\2019\Community\VC\Tools\MSVC\14.xx.xx\lib. Skopiuj plik .dll z folderu lib-vc2019 do folderu, w którym znajduje się plik wykonywalny Twojego projektu.

  7. Dodanie kodu do projektu:

    Dodaj kod do pliku main.cpp. Przykładowy kod z poprzednich odpowiedzi może być dobrym punktem startowym.

  8. Uruchomienie projektu:

    Skompiluj i uruchom projekt w Visual Studio.
